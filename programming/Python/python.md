# 1、python中共有属性和私有属性的区别

**a、公有属性可以继承而私有属性不可继承。**

**b、私有属性不可删除，读和写。**

**c、当私有属性添加了@property和@xxx.setter两个接口后与公有对比就只是*不能继承* 和*删除*了。**
